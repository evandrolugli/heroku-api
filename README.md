# Technologies
- node (runtime)
- typescript (language)
- fastify (web framework)
- prisma (database toolkit)

# Package Manager
- pnpm

> pnpm init
> pnpm i typescript @types/node tsx -D
> pnpm i fastify

> pnpm add prisma @prisma/client
> pnpm add -D prisma-cli
> pnpm prisma init

> pnpm prisma generate
> pnpm prisma migrate dev
